Components

POST /register
{

    "password": "password"
}

This component allows existing users to log in securely.


POST /login
{
    "username": "example",
    "password": "password"
}

Balance Checking

This component allows users to check their account balance.


GET /balance

Transaction History

This component allows users to view their transaction history.


GET /transactions

Usage

Instructions for using the banking application.

User Registration

To register a new user, send a POST request to the /register endpoint with the following JSON payload:


{
    "username": "example",
    "email": "example@example.com",
    "password": "password"
}

User Login

To log in an existing user, send a POST request to the /login endpoint with the following JSON payload:


{
    "username": "example",
    "password": "password"
}

Balance Checking

To check the account balance, send a GET request to the /balance endpoint.

Transaction History

To view the transaction history, send a GET request to the /transactions endpoint.
